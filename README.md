# Zhuoran Li Homepage

This repository contains the source for Zhuoran Li's academic homepage.

## Credits

The site structure is adapted from [Leonid Keselman's Jekyll fork](https://github.com/leonidk/leonidk.github.io), which is itself based on Jon Barron's academic website template.

## Local preview

If you have Jekyll installed locally, you can preview the site with:

```bash
jekyll serve
```
